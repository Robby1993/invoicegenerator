SETUP INSTRUCTIONS:

1. Create a new Flutter project:
   flutter create invoice_app
   cd invoice_app

2. Replace lib/main.dart with the code above

3. Create folder structure:
   - lib/models/
   - lib/providers/
   - lib/screens/
   - lib/utils/

4. Add dependencies to pubspec.yaml as shown at the top

5. Run:
   flutter pub get
   flutter run
